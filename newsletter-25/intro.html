השבוע אנחנו נכנסים עמוק יותר לניהול state בצד לקוח, ומגלים כמה מהלכים ספציפיים שיכולים לעשות הבדל ענק בקוד שלכם.

כי לפעמים כל מה שרציתם זה להציג כמה הודעות - ובסוף אתם בונים תשתית סנכרון שלמה 🤯

אז על מה נדבר?

איך TanStack DB מביא גישה חדשה לגמרי לאחסון נתונים בדפדפן, עם שאילתות SQL וזרימה בזמן אמת.
איך React Query Selectors יכולים לחסוך לכם רינדורים מיותרים ולשפר ביצועים משמעותית.
למה כדאי לחשב state ישירות מהשרת - במקום להחזיק אותו כפול ב-client.
ואם אתם עדיין עוטפים כל פונקציה ב-useCallback, אולי הגיע הזמן לעצור ולשאול למה בכלל.

כל מאמר כאן נוגע בנקודה כואבת מוכרת - ופותח פתרון מעשי.
